개인정보 처리방침

최종 업데이트: 2026년 6월 27일

밴드 역극 백업 확장프로그램 및 커뮤 백업 웹앱(이하 "서비스")은 사용자의 개인정보를 소중히 여기며, 다음과 같은 방침에 따라 처리합니다.

1. 수집하는 정보

서비스는 사용자가 직접 명시적인 동작(백업 버튼 클릭)을 한 경우에만 다음 정보를 임시로 처리합니다:

  • 사용자가 BAND에서 백업을 요청한 채팅방 또는 게시글의 텍스트 내용
  • 해당 콘텐츠의 작성자명, 시각, 캐릭터 정보 등 메타데이터

중요: 이 정보는 사용자의 로컬 브라우저 안에서만 처리됩니다. 외부 서버로 전송되지 않습니다.

2. 정보의 저장 위치

  • 확장프로그램 단독 사용 시: 백업한 데이터는 사용자가 지정한 로컬 폴더에 파일(TXT/JSON)로 저장되며, 확장프로그램 자체에는 어떤 데이터도 보관되지 않습니다.
  • 커뮤 백업 웹앱 연동 시: 사용자가 명시적으로 "정돈소로 보내기" 버튼을 누른 경우에만, 백업 데이터가 사용자의 브라우저 localStorage에 저장됩니다. 이 데이터는 사용자의 디바이스를 떠나지 않습니다.

3. 정보의 송신 여부

서비스는 어떠한 외부 서버에도 사용자의 콘텐츠나 개인정보를 전송하지 않습니다. 분석 도구, 광고 네트워크, 트래킹 픽셀 일체 사용하지 않습니다.

유일한 데이터 전송은 사용자가 직접 트리거하는 확장프로그램 ↔ 커뮤 백업 웹앱 간의 같은 브라우저 내 postMessage 통신이며, 이는 사용자의 디바이스 외부로 나가지 않습니다.

4. 권한 사용 내역

  • activeTab: 사용자가 백업 버튼을 누른 BAND 탭의 콘텐츠에 접근
  • scripting: BAND 페이지에 백업 UI를 표시
  • downloads: 백업 파일을 로컬에 저장
  • host_permissions (band.us): BAND 도메인에서만 동작

다른 웹사이트나 시스템 자원에는 일절 접근하지 않습니다.

5. 데이터 삭제

웹앱에 저장된 데이터는 브라우저의 localStorage를 비우거나 사이트 데이터를 삭제함으로써 즉시 제거할 수 있습니다. 웹앱 내의 각 세션 카드에서 삭제 버튼으로 개별 삭제도 가능합니다.

6. 제3자 제공

서비스는 사용자의 어떠한 정보도 제3자에게 제공하거나 판매하지 않습니다.

7. 미성년자 보호

서비스는 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. BAND 자체의 이용 약관 및 연령 제한을 따릅니다.

8. 방침 변경

개인정보 처리방침이 변경될 경우, 이 페이지에 변경된 내용과 일자를 명시합니다.

9. 문의

개인정보 처리에 관한 문의는 GitHub 이슈 또는 확장프로그램 개발자에게 연락해주세요.
